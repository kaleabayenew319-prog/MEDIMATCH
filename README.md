The 75% Confidence Rule (Core Innovation)

| Condition | Action |
|-----------|--------|
| ≥75% confidence + serious | Hospital Path – Medical report + specialist matching + Telebirr booking |
| <75% confidence OR minor illness | Pharmacy Path – Deposit (30 ETB) + QR Pharmacist Slip + fulfillment |
| Red flag detected | Emergency Bypass – Immediate 8335 call + nearest hospital |

Context Memory
•	Remembers past diagnoses (≥75% confidence) → no redundant questions
•	Detects recurring symptom patterns → flags chronic conditions
•	Stores allergies permanently → safety critical
•	Patient-controlled consent and deletion
